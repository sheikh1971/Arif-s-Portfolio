# **App Name**: SynapticFolio

## Core Features:

- Dynamic Content Presentation: Render all portfolio content (Hero, About, Experience, Projects, Skills, Education, Impact, Contact) across highly structured and clearly defined sections.
- Contact Messaging System: Provide a functional contact form allowing visitors to send messages directly.
- Resume Download Functionality: Enable visitors to download the applicant's resume with a dedicated button.
- Project Filtering & Tagging: Implement interactive filters for project cards based on defined tags (e.g., AI, CV, Embedded, Mobile).
- Theme Toggle (Dark/Light Mode): Allow users to switch between a default dark premium theme and a light mode.
- Responsive & Adaptive Design: Ensure the website layout and content adapt seamlessly to various screen sizes and devices, from mobile to desktop.
- AI Project Insights Tool: A generative AI tool that, on demand, provides visitors with concise summaries or highlighted technical insights for selected projects within the portfolio.

## Style Guidelines:

- The chosen color palette is anchored in a futuristic AI engineering vibe, leveraging a base of cool blue, purple, and cyan tones suitable for a professional yet innovative persona. The default dark premium theme prioritizes strong contrast and clarity.
- Primary color: #A778F7 (a bright medium purple), selected for its vibrant yet composed feel, contrasting effectively against a dark background and aligning with modern tech aesthetics. (HSL: 250, 60%, 70%)
- Background color: #1C1B22 (a very dark purplish-grey), offering a deep, sophisticated canvas that makes lighter elements pop and supports the 'dark premium' theme. (HSL: 250, 15%, 12%)
- Accent color: #48B1F4 (a bright blue), used to draw attention to interactive elements, CTAs, and key highlights, providing an energetic counterpoint to the primary purple and tying into the requested blue/cyan gradients. (HSL: 220, 70%, 60%)
- Headlines and prominent text will use 'Space Grotesk' (sans-serif) for its modern, tech-focused, and slightly scientific aesthetic, emphasizing the engineering credibility. Body text will utilize 'Inter' (sans-serif), a clear, objective font that ensures high readability and structured clarity for detailed information blocks.
- Modern, minimalist geometric icons will be used throughout the site for navigation, social links, and section indicators, maintaining a clean and tech-savvy aesthetic without visual clutter.
- Inspired by leading platforms like Apple and Stripe, the layout features highly structured sections, clean lines, a prominent split-screen hero section, and interactive glassmorphism cards for projects, enhancing visual depth with subtle shadows.
- Subtle scroll-based reveal animations will introduce content as the user navigates, complemented by smooth scrolling and a very subtle, unobtrusive AI/neural network inspired background animation to reinforce the thematic elements.
